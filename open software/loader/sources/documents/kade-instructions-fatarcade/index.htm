Programming Instructions

  1. Connect jumper to both pins on the KADE fatArcade. The jumper can be found next to the white reset button.

  2. If KADE fatArcade is already connected to computer then press the white reset button otherwise connect to computer via USB.

  3. Select a firmware from the list of those available.

  4. Click the “Program KADE with Selected Firmware” button.
    The firmware should be updated in a few seconds.

  5. Remove the jumper and press the white reset button. Note that the jumper can be connected to just one pin for safe keeping.

  6. KADE fatArcade is now ready to be used as an encoder.


For further information please refer to the quick start guide and the kadevice.com web site.
