Trackball/Spinner to USB Mouse and Keyboard Encoder (Beta)

pin Function
A1 Mouse X Axis - Phase A
A2 Mouse X Axis - Phase B
A3 Mouse Y Axis - Phase A
A4 Mouse Y Axis - Phase B
A5 Mouse Z Axis (Scroll Wheel) - Phase A
A6 Mouse Z Axis (Scroll Wheel) - Phase B
A7 Mouse Button 1 (Left Click)
A8 Mouse Button 2 (Right Click)
A9 Mouse Button 3 (Middle Click)
A10 ESCAPE Key (Exit)
B1 Left-Control Key (P1 Button 1)
B2 Left-Alt Key (P1 Button 2)
B3 Space Key (P1 Button 3)
B4 Left-Shift Key (P1 Button 4)
B5 Z Key (P1 Button 5)
B6 X Key (P1 Button 6)
B7 5 Key (P1 Enter Coin)
B8 1 Key (P1 Start)
B9 ENTER Key (Select)
B10 P Key (Pause)
HWB Adjust the mouse axis sensitivty.
+5V +5V Power
RST Reset
Ground Ground

Notes
Connect signal wires from trackball and/or spinner to the phase inputs on the KADE encoder.
There are 2 connections per axis - phase A and phase B.

For e.g. A Happ Trackball has green and yellow wires connected to A1 and A2 for X Axis. Purple and blue are connected to A3 and A4 for Y Axis. Finally the red and black are connect to power and ground.
